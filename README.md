# Flagship Builder

A vanilla JavaScript, SCSS, and HTML project with Vite build system.

## 프로젝트 구조

```
flagship-builder/
├── src/
│   ├── styles/
│   │   └── style.scss
│   └── main.js
├── public/
├── index.html
├── vite.config.js
├── package.json
└── README.md
```

## 기능

- Vanilla JavaScript를 사용한 웹 개발
- SCSS를 활용한 스타일링
- Vite 빌드 시스템

## 설치 방법

1. 저장소 클론

```bash
git clone https://github.com/khj0311/flagship-builder.git
cd flagship-builder
```

2. 의존성 설치

```bash
npm install
```

3. 개발 서버 시작

```bash
npm run dev
```

## 빌드 방법

프로덕션용 빌드:

```bash
npm run build
```

빌드된 결과물 미리보기:

```bash
npm run preview
```

## 기술 스택

- HTML5
- SCSS
- JavaScript (ES6+)
- Vite (빌드 툴)
